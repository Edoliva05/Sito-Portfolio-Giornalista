# Portfolio Giornalista - Agostino Lenzi

Un sito web portfolio elegante, dinamico e completamente responsive, sviluppato per un amico giornalista. Il sito permette di presentare la propria biografia, fornire contatti diretti, far scaricare il curriculum vitae e mostrare una galleria sempre aggiornata dei propri articoli cartacei e digitali.

---

## Funzionalità Principali

* **Gestione Dinamica degli Articoli:** Grazie all'integrazione con **Sanity.io**, gli articoli possono essere aggiunti, modificati o rimossi tramite un comodo pannello di controllo, senza dover mettere mano al codice sorgente.
* **Design:** Un'interfaccia utente pulita, con una palette colori studiata per non affaticare la vista e richiamare l'estetica della carta stampata.
* **Responsività:** Layout fluido e moderno costruito con CSS Grid e Flexbox, che si adatta perfettamente a qualsiasi dispositivo (smartphone, tablet, desktop).
* **Routing con JS:** Generazione dinamica dei contenuti tramite JavaScript, per caricare i dettagli del singolo articolo in modo rapido e leggero.

---

## Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** [Sanity.io](https://www.sanity.io/)
* **Version Control:** Git & GitHub
